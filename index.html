한국, 필리핀, 베트남 3자 화상회의를 위한 전용 입장 포털입니다. 아래 입장 버튼을 선택하여 지정된 회의방으로 바로 접속할 수 있습니다.

이 페이지는 VDO.Ninja 회의방 접속을 중계하며, 실제 영상 연결은 VDO.Ninja에서 처리됩니다. 카메라 줌·PTZ·트래킹 등 장비 제어는 OBSBOT Center 또는 OBSBOT 장비 자체 기능에서 수행합니다.

⚠ 새 창 차단 설정이 활성화되어 있으면 입장 창이 열리지 않을 수 있습니다. 브라우저 팝업 차단을 해제해 주세요.

회의 전 점검 및 문제 해결

회의 시작 5분 전에 확인해 주세요.

회의 전 체크리스트

  • OBSBOT 카메라를 USB로 연결했는지 확인
  • Chrome 또는 Edge 브라우저 사용
  • 입장 후 카메라 목록에서 OBSBOT 선택
  • 마이크 중복 선택 여부 확인
  • 이어폰 또는 회의용 스피커폰 사용 권장
  • 가능하면 유선 인터넷 사용
  • 불필요한 프로그램 종료

문제 발생 시 조치방법

  • 영상이 안 나올 때브라우저 카메라 권한 확인
  • OBSBOT이 안 보일 때USB 연결 및 UVC 모드 확인
  • 소리가 울릴 때이어폰 사용 또는 스피커 볼륨 조정
  • 끊김이 심할 때Wi-Fi 대신 유선 인터넷 사용
  • 접속이 안 될 때vdo.ninja 접속 가능 여부 확인
